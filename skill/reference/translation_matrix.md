# German->English Translation Matrix

Key Freudian terms where translation obscures the original meaning.
Understanding the German originals deepens the archetype mappings.

| German | Strachey English | Literal Meaning | Archetype Impact |
|--------|-----------------|-----------------|------------------|
| Das Es | Id | "The It" | The impersonal, uncontrollable force -- like "it is raining." Agents should treat raw impulses as impersonal system events, not intentional acts. |
| Das Ich | Ego | "The I" | The everyday self. The orchestrator agent is not a special authority -- it is simply the agent's sense of "I am doing this." |
| Das Uber-Ich | Superego | "The Over-I" | An authority *above* the self. Guardrails should feel like external authority, not self-censorship. |
| Trieb | Drive (not Instinct) | "Push, urge" | A continuous psychological push, not a rigid biological reflex. Agent goals are drives, not instincts -- they adapt. |
| Besetzung | Cathexis | "Occupation, investment" | Like investing troops or capital. Attention allocation is *investment* -- it has costs and returns. Maps to **fixation** archetype. |
| Verdrangung | Repression | "Pushing away" | Active pushing, not passive forgetting. Context pruning is an active operation. Maps to **censor-gate** archetype. |
| Verschiebung | Displacement | "Shifting" | Shifting emotional charge. When agents redirect effort, the *energy* (priority) shifts too. Maps to **dream-work** archetype. |
| Verdichtung | Condensation | "Thickening, compression" | Making denser, not smaller. Synthesis should *enrich*, not merely summarize. Maps to **dream-work** archetype. |
| Widerstand | Resistance | "Standing against" | Active opposition, not passive friction. Failures *oppose* the agent's approach for structural reasons. Maps to **freudian-slip** archetype. |
| Fehlleistung | Parapraxis | "Faulty achievement" | An action that *achieves* something -- just not what was consciously intended. Errors have meaning. Maps to **freudian-slip** archetype. |
| Wiederholungszwang | Repetition Compulsion | "Compulsion to repeat" | Zwang (compulsion) implies involuntary force. Loop detection fights a systemic tendency, not a bug. |
| Traumarbeit | Dream-Work | "Dream work" | The entire transformation pipeline: condensation, displacement, and secondary revision operating together. |
| Sekundare Bearbeitung | Secondary Revision | "Secondary working-over" | Context curation constructs narrative, not mere filtering. The final dream-work operation imposes coherence. Maps to **dream-work** archetype. |
| Psychischer Apparat | Psychic Apparatus | "Mental apparatus" | Not metaphor -- Freud meant a processing device with distinct stations. A system of agencies, not a reflex arc. Maps to **ephemeral** archetype. |
| Traumelemente | Dream Elements | "Dream elements" | Ephemeral by nature -- they appear during sleep, perform work, and dissolve on waking. Maps to **ephemeral** archetype. |
| Instanz | Agency (not Instance) | "Judicial authority" | Each psychic structure is a *tribunal* making judgments. Agent layers have authority, not just function. |
| Unbehagen | Discontent | "Deep unease, malaise" | Not simple dissatisfaction. The chronic tension between agent capability and safety constraints. |
| Traumdeutung | Dream Interpretation | "Dream meaning-making" | Deutung emphasizes *making meaning*, not just interpreting. Agents don't just parse -- they construct understanding. |
